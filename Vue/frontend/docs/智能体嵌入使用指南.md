# 智能体嵌入使用指南

## 功能概述

智能体嵌入功能为教师端的作业管理和考试管理模块提供了 AI 智能助手支持。基于 Dify 平台的聊天机器人技术，提供专业的教学辅助服务。

## 功能特性

### 🤖 智能对话
- 实时聊天界面
- 自然语言交互
- 上下文理解能力

### 📝 作业批改辅助
- 智能批改建议
- 自动评分参考
- 详细反馈生成

### 📊 考试管理助手
- 试卷创建建议
- 考试分析报告
- 成绩统计辅助

### 💬 专业问答
- 教学方法建议
- 课程设计指导
- 教育相关咨询

## 使用方法

### 1. 在作业管理页面

1. **访问页面**：进入教师端 → 作业管理
2. **显示助手**：点击页面顶部的"显示智能助手"按钮
3. **开始对话**：点击右下角的蓝色聊天按钮
4. **提问互动**：在聊天窗口中输入您的问题

### 2. 在考试管理页面

1. **访问页面**：进入教师端 → 考试管理
2. **显示助手**：点击页面顶部的"显示智能助手"按钮
3. **开始对话**：点击右下角的蓝色聊天按钮
4. **咨询问题**：询问考试相关的问题

### 3. 在 AI 助手专页

1. **访问页面**：进入教师端 → AI 智能助手
2. **查看功能**：了解智能助手的各项功能
3. **开始使用**：按照使用指南进行操作

## 常见问题

### Q1: 聊天按钮没有显示？
**答案**：检查网络连接，确保能够访问 Dify 平台（http://219.216.65.108）

### Q2: 智能助手回答不准确？
**答案**：
- 尝试更详细地描述问题
- 提供更多的上下文信息
- 使用教育专业术语

### Q3: 聊天窗口无法打开？
**答案**：
- 刷新页面重试
- 检查浏览器是否阻止了弹窗
- 确认 JavaScript 已启用

### Q4: 如何隐藏智能助手？
**答案**：点击页面顶部的"隐藏智能助手"按钮

## 使用建议

### 提问技巧
1. **具体明确**：详细描述问题和需求
2. **提供上下文**：说明教学场景和背景
3. **分步询问**：将复杂问题分解为多个小问题

### 功能应用
1. **作业批改**：询问批改标准和评分建议
2. **考试设计**：请求试卷结构和题目建议
3. **教学方法**：咨询教学策略和方法
4. **数据分析**：请求成绩分析和报告建议

### 最佳实践
1. **定期使用**：形成使用习惯，提高教学效率
2. **反馈优化**：根据回答质量调整提问方式
3. **结合实际**：将 AI 建议与实际教学经验结合

## 注意事项

### 隐私保护
- 不要输入学生的个人敏感信息
- 避免分享机密的教学资料
- 注意保护考试内容的安全性

### 使用限制
- 智能助手的建议仅供参考
- 最终决策应由教师自主判断
- 重要决定需要人工审核确认

### 技术要求
- 需要稳定的网络连接
- 支持现代浏览器
- JavaScript 需要启用
